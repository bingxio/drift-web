Install


First pull the project to the local.


> git clone https://github.com/drift-lang/drift.git


· Switch CC field in Makefile and specify your g++ compiler path

· The development version is 10.2.0


> make

> make install


The drift executable file will be generated in the project directory.


> file drift


Set its running path

please ensure that the std standard library folder exists in the directory.


> export PATH=$PWD:$PATH


Run


> drift # REPL MODE

> drift [ft file] # FILE MODE


> drift -d # REPL AND DEBUG MODE

> drift [ft file] -d # FILE AND DEBUG MODE


> drift -b # REPL AND DIS BYTECODE

> drift [ft file] -b # FILE AND DIS BYTECODE
